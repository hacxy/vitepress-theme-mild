 vitepress-theme-mild

<h4 align="center">
遵从 Vitepress 本质的简约风格博客主题.
</h4>

![NPM Version](https://img.shields.io/npm/v/vitepress-theme-mild)

- 💡 一切从简, 使用简单, 配置简单, 风格简约
- 🌈 一些过渡动画和页面加载进度条
- 📃 自动扫描文章列表, 文件发生变化时同步热更新🔥
- ✨ 根据文章分类自动生成侧边栏分组
- 🏷️ 标签页布局
