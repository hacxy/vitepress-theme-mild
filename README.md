<h1 align="center">
 vitepress-theme-mild
</h1>

<h4 align="center">
极简风格的 Vitepress 博客主题.
</h4>

![NPM Version](https://img.shields.io/npm/v/vitepress-theme-mild)

- 💡 简约风格且配置简单
- 🌈 过渡动画效果以及页面加载进度条
- 📃 自动扫描根目录生成文章列表与分页器
- ✨ 根据文章分类自动生成侧边栏分组
