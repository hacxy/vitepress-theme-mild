# vitepress-theme-mild

极简风格的 Vitepress 博客主题.

- 💡 简约风格且配置简单
- 🌈 过渡动画效果以及页面加载进度条
- 📃 自动扫描根目录生成文章列表与分页器
- ✨ 根据文章分类自动生成侧边栏分组
