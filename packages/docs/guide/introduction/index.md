# 概述

`VitePress Theme Mild` 是一个使用简单且具备丰富能力的 VitePress 主题, 它基于默认主题开发, 并额外提供了博客布局, 帮助你快速搭建个人博客或者技术文档.

## 为什么开发这个主题

[开发VitePress主题的初衷](https://hacxy.cn/docs/posts/dev-vitepress-theme/1.start.html) 这篇文章详细记录了我为什么开发这个主题, 以及未来的规划
